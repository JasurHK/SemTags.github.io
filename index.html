HTML semantic tags

What is semantic tags?

HTML semantic tags provide meaning to the web content, making it easier for both search engines and developers to understand the structure and content of the web page. These tags also improve accessibility and SEO. Here are table of the key semantic tags in HTML5

June 27, 2024

Semantic tags

Tag Name Description
<header> Header An introductory content, typically a group of introductory or navigational aids
<nav> Navigation A section of a page that links to other pages or parts within the page
<main> Main The main content of the document, which is unique to the document
<section> Section A standalone section of content that can be used in documents and applications
<article> Article A self-contained composition in a document, page, or site, which is intended to be independently distributable or reusable
<aside> Aside A portion of a document whose content is only indirectly related to the document's main content
<footerr> Footer The footer for its nearest sectioning content or sectioning root element
